# 🚀 Sarath V P Portfolio

A minimal, high-performance, and attractive personal portfolio built with the **MERN** stack and **Vite**. This portfolio is designed to showcase my expertise in full-stack development, my projects, and my professional skills.

## ✨ Features

- 🌑 **Dark Mode by Default**: A sleek, modern black theme with vibrant `#39FF14` (Neon Green) accents.
- 📱 **Fully Responsive**: Optimized for all screen sizes, from mobile to ultra-wide displays.
- 🛠️ **Real-time Projects**: Showcases live projects from GitHub with detailed tech stacks and descriptions.
- ⚡ **Vite-powered**: Blazing fast development and build times.
- 🎨 **Tailwind CSS v4**: Modern styling with maximum flexibility.
- 📧 **Direct Contact**: Integrated email messaging and social links (GitHub, LinkedIn, LeetCode).

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📂 Projects Featured

1. **learnCircle (LMS)**: A mentor-learner connection platform with real-time video calls and community forums.
2. **Inventory Management System**: A full-stack system for inventory, customer, and sales tracking.
3. **Task Management Platform**: Feature-rich task management with real-time updates and analytics dashboard.
4. **ZooCart**: An e-commerce platform for mobile phones featuring Google OAuth and Razorpay integration.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sarath2308/portfolio-sarath.git
   cd portfolio-sarath/portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
